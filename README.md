# POO — Interfaces en Java

Proyecto diseñado para practicar **interfaces**, **polimorfismo** y **abstracción** mediante figuras geométricas que implementan la interfaz `FiguraGeometrica`.

## Características
- Interfaz `FiguraGeometrica` con el método `calcularArea()`.
- Implementaciones:
  - Cuadrado
  - Rectángulo
  - Triángulo
  - Rombo
  - Romboide
  - Trapecio
- Ejecución dinámica desde `Main.java` para demostrar polimorfismo.

## Estructura
src/org.generation.Interfaces/
├─ FiguraGeometrica.java
├─ Cuadrado.java
├─ Rectangulo.java
├─ Triangulo.java
├─ Rombo.java
├─ Romboide.java
├─ Trapecio.java
└─ Main.java

## Tecnologías
- Java 8+
- Eclipse IDE

## Ejecución
Ejecuta `Main.java` para probar varias figuras polimórficamente.
